# Azure Microservices DevOps Project

End-to-end cloud-native microservices project using:

- React frontend
- Node.js backend
- PostgreSQL
- Docker
- Kubernetes (AKS)
- Terraform
- Azure DevOps
- Azure Key Vault
- Azure Container Registry
- Autoscaling
- Multi-environment architecture

## Architecture

Frontend → Backend → PostgreSQL
